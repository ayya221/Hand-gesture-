Aero Gesture Control

Touchless human-computer interaction using hand gestures, computer vision, and formal state machines.

DFA State Machine
State 1 ⬤
IDLE
Waiting for input
→
State 2 👁
DETECTING
Gesture spotted
→
State 3 ⏳
CONFIRMING
Held for N frames
→
State 4 ⚡
EXECUTING
Action fired!
→
State 5 🔄
COOLDOWN
Debounce period

Click a state to preview it · The DFA ensures no false-positive triggers

Real-World Applications
🏥
Hospital & Sterile Rooms
Surgeons control displays without touching surfaces, reducing cross-contamination risk during procedures.
🏫
Smart Classrooms
Teachers control slides and draw on virtual boards while facing students — no remote needed.
♿
Accessibility
People with motor impairments or limited mobility can interact with computers via natural hand movements.
🏭
Industrial Control
Factory workers wearing thick gloves can control dashboards and machinery without removing safety gear.
🎮
Gaming & XR
Natural gesture input for AR/VR environments where physical controllers break immersion.
🚗
Automotive HUDs
Drivers interact with dashboard systems without taking eyes off the road — safer in-car UX.
Setup & Install
# Install dependencies
pip install opencv-python mediapipe pyautogui numpy

# Run the system
python aero_gesture_control.py

# Project structure
aero_gesture_control.py
├── GestureDFA         # State machine (IDLE→DETECTING→CONFIRMING→EXECUTING→COOLDOWN)
├── GestureClassifier  # MediaPipe landmark → gesture name
├── AirDrawing         # Glowing neon trail renderer
├── ActionExecutor     # Maps gestures → keyboard/mouse actions
└── draw_hud()         # Real-time on-screen display

# Gestures supported
☝  INDEX UP    →  Air draw mode (instant, no DFA)
🖐  ALL UP      →  Clear canvas
👍  THUMB UP    →  Next slide (→ key) [DFA confirmed]
✌  3 FINGERS   →  Prev slide (← key) [DFA confirmed]
🤏  PINCH       →  Mouse click         [DFA confirmed]
✊  FIST        →  Space / pause       [DFA confirmed]
